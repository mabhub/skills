# Phrases à supprimer

## Ouvertures pompeuses

Supprimer ces formules d'annonce. Énoncer directement le contenu.

- "Il convient de noter que"
- "Il convient de souligner que"
- "Il convient de rappeler que"
- "Il convient de préciser que"
- "Il est important de souligner que"
- "Il est essentiel de comprendre que"
- "Il est intéressant de noter que"
- "Il est à noter que"
- "Il va sans dire que"
- "Force est de constater que"
- "Notons que" / "Soulignons que" / "Rappelons que"
- "Il est indéniable que"
- "Il est clair que"
- "Il apparaît que" / "Il apparaît clairement que"
- "On ne saurait ignorer que"
- "On ne peut que constater que"
- "Il n'est pas inutile de rappeler que"
- "Comme chacun le sait"
- "Comme nous le savons"
- "Nul ne saurait contester que"
- "Il faut bien reconnaître que"
- "À bien y regarder"
- "Tout d'abord, il convient de"
- "En premier lieu, il est important de"
- "Avant toute chose"
- "Il est primordial de"
- "Il est fondamental de comprendre que"

Toute construction "Il convient de + verbe + que" est du remplissage. Couper et énoncer le fait.

## Béquilles d'emphase

N'ajoutent aucun sens. Les supprimer.

- "Cela ne fait aucun doute."
- "C'est un fait."
- "C'est indéniable."
- "Cela mérite d'être souligné."
- "C'est là que le bât blesse."
- "Et ce n'est pas un hasard."
- "Ce n'est pas anodin."
- "C'est tout sauf anodin."
- "Cela en dit long."
- "La nuance est de taille."
- "L'enjeu est de taille."
- "Et pour cause."
- "C'est là tout le paradoxe."
- "C'est là toute la question."
- "C'est bien là le problème."

## Jargon corporate

Remplacer par du langage courant.

| Éviter | Utiliser |
|--------|----------|
| Dans un contexte de | (couper, souvent inutile) |
| À l'aune de | Selon, d'après |
| Au regard de | Selon, vu |
| En termes de | (reformuler : "les coûts" au lieu de "en termes de coûts") |
| À travers le prisme de | Selon, du point de vue de |
| Dans une logique de | Pour |
| Dans une optique de | Pour |
| Dans une démarche de | Pour |
| Mettre en œuvre | Faire, appliquer |
| Être en capacité de | Pouvoir |
| Être en mesure de | Pouvoir |
| Optimiser | Améliorer (sauf contexte technique précis) |
| Valoriser | (dire ce qu'on fait réellement) |
| Mutualiser | Partager |
| Capitaliser sur | Utiliser, tirer parti de |
| Accompagner la transformation | (dire quel changement précis) |
| Co-construire | Construire ensemble, élaborer |
| Monter en compétence | Apprendre, se former |
| Piloter (sens corporate) | Gérer, diriger |

## Adverbes et locutions béquilles

Traquer et supprimer quand ils n'apportent aucune information. Conserver un adverbe seulement s'il change le sens de la phrase.

Adverbes en -ment coupables par défaut :

- "effectivement"
- "véritablement"
- "réellement"
- "indéniablement"
- "incontestablement"
- "fondamentalement"
- "intrinsèquement"
- "indubitablement"
- "assurément"
- "nécessairement"
- "inévitablement"
- "manifestement"
- "inéluctablement"
- "considérablement"
- "significativement"
- "substantiellement"
- "notablement"
- "sensiblement"
- "profondément"
- "pleinement"

Locutions adverbiales béquilles :

- "à bien des égards"
- "dans une large mesure"
- "dans une certaine mesure"
- "tout à fait"
- "en grande partie"
- "à juste titre"
- "à plus d'un titre"
- "en tout état de cause"
- "en l'occurrence"
- "de facto" (sauf texte juridique)
- "à proprement parler"
- "au sens propre du terme"
- "ni plus ni moins"

Note : contrairement à l'anglais (stop-slop bannit tous les adverbes), le français tolère les adverbes informatifs ("mensuellement", "localement", "manuellement"). Bannir les adverbes d'emphase vide — ceux qui intensifient sans informer.

## Connecteurs logiques en excès

Les LLM en français empilent les connecteurs. Règle : un connecteur est nécessaire quand le lien logique entre deux phrases n'est pas évident sans lui. Si le lien est évident, le connecteur est du bruit. Maximum deux connecteurs par paragraphe.

Connecteurs suremployés :

- "En effet"
- "Par ailleurs"
- "De surcroît"
- "En outre"
- "Qui plus est"
- "De plus"
- "Par conséquent"
- "Ainsi"
- "Dès lors"
- "De ce fait"
- "C'est pourquoi"
- "Toutefois"
- "Néanmoins"
- "Cependant"
- "Or"
- "En revanche"
- "À cet égard"
- "À ce titre"
- "Il n'en demeure pas moins que"
- "Force est de reconnaître que"

## Métacommentaires

Le texte doit avancer, pas s'annoncer. Supprimer les commentaires autoréférents.

- "Comme nous allons le voir"
- "Comme nous l'avons vu"
- "Comme mentionné précédemment"
- "Il est intéressant de noter que"
- "Il est important de mentionner que"
- "Il est à souligner que"
- "Nous verrons plus loin que"
- "La suite de cet article montre que"
- "Comme évoqué en introduction"
- "Dans la section suivante"
- "Penchons-nous sur"
- "Attardons-nous sur"
- "Examinons maintenant"
- "Intéressons-nous à"
- "Tournons-nous vers"
- "Voyons maintenant"
- "Passons maintenant à"
- "Abordons à présent"

## Emphase performative

Fausse sincérité et fausse franchise :

- "Il faut bien l'admettre"
- "Reconnaissons-le"
- "Avouons-le"
- "Disons-le clairement"
- "Soyons honnêtes"
- "Soyons clairs"
- "Sans ambiguïté"
- "Sans détour"
- "À vrai dire"
- "En toute franchise"
- "En toute objectivité"

## Déclaratifs vagues

Phrases qui annoncent une importance sans la montrer. Les couper ou remplacer par le fait précis.

- "Les enjeux sont considérables"
- "Les conséquences sont majeures"
- "Les implications sont profondes"
- "Cette question est fondamentale"
- "Ce sujet est crucial"
- "L'impact est significatif"
- "Les défis sont nombreux"
- "Les possibilités sont infinies"
- "Le potentiel est immense"
- "Les raisons sont multiples"
- "Les facteurs sont nombreux"
- "Les perspectives sont prometteuses"
- "Les résultats sont encourageants"
- "La problématique est complexe"

Si une phrase dit que quelque chose est important/crucial/fondamental sans montrer la chose précise, la couper ou la remplacer par le fait.
