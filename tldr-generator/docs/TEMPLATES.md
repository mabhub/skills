# Templates de TL;DR par type d'article

Ce fichier contient des exemples de structures de TL;DR pour différents types d'articles.

## Template : Article technique/documentation

```markdown
---
created: [date originale]
tags: [mots-clés techniques pertinents]
source: [URL]
author: [auteur]
type: documentation technique
themes: [thèmes principaux]
---

# [Titre] - TL;DR

## Résumé exécutif

[2-3 paragraphes résumant la technologie/concept présenté, son objectif principal, et son impact/utilité]

## Présentation de [technologie/concept]

[Description de ce dont il s'agit, contexte de création/utilisation]

**Caractéristiques principales :**
- Point clé 1
- Point clé 2
- Point clé 3

## Architecture et fonctionnement

[Explication du fonctionnement, avec sous-sections si nécessaire]

### [Composant/Aspect 1]
[Détails]

### [Composant/Aspect 2]
[Détails]

## Cas d'usage et avantages

**Applications :**
- Usage 1
- Usage 2
- Usage 3

**Avantages clés :**
- Bénéfice 1
- Bénéfice 2

## Limitations et considérations

[Points d'attention, contraintes, limitations connues]

[Section optionnelle sur disponibilité, prérequis, etc.]
```

## Exemples concrets de bon formatage

### Front-matter enrichi - Exemple

```yaml
---
created: 2026-02-02T15:53:47 (UTC +01:00)
tags: [IA, agents IA, Moltbook, R&D IA, productivité, robotique, émulation cérébrale]
source: https://importai.substack.com/p/import-ai-443
author: Jack Clark
type: newsletter
themes: [écologie d'agents IA, automatisation recherche IA, impact économique IA]
---
```

### Listes à puces vs paragraphes - Équilibre

**❌ Trop haché :**
```markdown
- Article parle de Moltbook
- C'est un réseau social
- Pour agents IA
- Dizaines de milliers d'agents
- Discutent entre eux
```

**✅ Équilibré :**
```markdown
**Moltbook** représente une innovation majeure : un réseau social type Reddit 
dédié aux agents IA. Cette plateforme héberge des dizaines de milliers d'agents 
qui discutent, collaborent et créent du contenu de manière autonome.

**Caractéristiques principales :**
- Réseau social dédié aux agents IA
- Intégration avec OpenClaw pour accès système complet
- Conversations autonomes entre agents sans supervision humaine
```

### Préservation des informations - Exemple

**Article original :**
> "Since early 2024, our performance engineering team has used a take-home test 
> where candidates optimize code for a simulated accelerator. Over 1,000 candidates 
> have completed it, and dozens now work here."

**✅ TL;DR qui préserve :**
```markdown
Depuis début 2024, l'équipe d'ingénierie performance utilisait un test 
d'optimisation de code pour accélérateurs simulés. Plus de 1000 candidats 
l'ont passé, avec des dizaines de recrutements.
```

**❌ TL;DR qui perd des infos :**
```markdown
Anthropic utilisait un test de code pour recruter.
```
